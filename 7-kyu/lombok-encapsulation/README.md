<h2><a href=https://www.codewars.com/kata/5a03295680171ffd7b0000c7/train/java target="_blank">Lombok Encapsulation</a></h2><h3>7 kyu</h3><p>In Java, we have a very powerful library named Lombok. It helps to reduce the boilerplate code.</p><p>This Kata is a challenge created for you to learn how to use <a href="https://projectlombok.org/" data-turbolinks="false" target="_blank">lombok</a>.</p><p>Giving the following class:</p><pre><code>public class EncapsulationDemo{  public int number;  public String stringValue;  public Object anObject;}</code></pre><p>We have to do the following modifications to properly encapsulate info:</p><ul><li>make fields private</li><li>provide accessors to the internal fields ( setters/getters )</li><li>provide an empty constructor to support Java Beans specification</li><li>Add a constructor that can receive parameters for all of the fields</li></ul><p>Note: make sure if you don't use Lombok to enforce proper naming conventions.</p><p>If you want a harder version please try: <a href="https://www.codewars.com/kata/5e5cc5aa6760d70036ea6d22/java" data-turbolinks="false" target="_blank">Lombok Encapsulation II</a></p>
