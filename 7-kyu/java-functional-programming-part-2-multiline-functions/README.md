<h2><a href=https://www.codewars.com/kata/54a6c6e7478d8ee02d000cd9/train/java target="_blank">Java Functional Programming (Part 2: Multiline Functions)</a></h2><h3>7 kyu</h3><p>With the release of Java 8, Java has finally added support for "lambda functions", that is, variables that contain a function which can operate on data just as class methods can (well, not <em>just</em> as class methods can...)</p><p>Let's learn about lambdas.</p><p>This is part 2 of a 4 part series; the other katas are:</p><ul><li>Part 1: <a href="http://www.codewars.com/kata/java-functional-programming-part-1-the-beginning/java" data-turbolinks="false" target="_blank">http://www.codewars.com/kata/java-functional-programming-part-1-the-beginning/java</a></li><li>Part 3: <a href="http://www.codewars.com/kata/java-functional-programming-part-3-closured-for-business" data-turbolinks="false" target="_blank">http://www.codewars.com/kata/java-functional-programming-part-3-closured-for-business</a></li><li>Part 4: <a href="http://www.codewars.com/kata/java-functional-programming-part-4-row-row-row-your-boat-gently-down-the-dot-dot-dot" data-turbolinks="false" target="_blank">http://www.codewars.com/kata/java-functional-programming-part-4-row-row-row-your-boat-gently-down-the-dot-dot-dot</a></li></ul><p>Last time, we did something similar to this:</p><pre><code class="language-java"><span class="cm-variable">Function</span><span class="cm-operator">&lt;</span><span class="cm-variable">MyObject</span>, <span class="cm-type">String</span><span class="cm-operator">&gt;</span> <span class="cm-variable">f</span> <span class="cm-operator">=</span> <span class="cm-variable">p</span> <span class="cm-operator">-&gt;</span> <span class="cm-variable">p</span>.<span class="cm-variable">toString</span>();<span class="cm-type">String</span> <span class="cm-variable">myString</span> <span class="cm-operator">=</span> <span class="cm-variable">f</span>.<span class="cm-variable">apply</span>(<span class="cm-variable">myObject</span>); <span class="cm-comment">//Stores whatever the toString() of myObject is in myString</span></code></pre><p>But what if we need more than one line in our function for the calculations? The syntax to do so is to enclose the function's commands in curly brackets and include a return statement:</p><pre><code class="language-java"><span class="cm-variable">Function</span><span class="cm-operator">&lt;</span><span class="cm-variable">MyObject</span>, <span class="cm-type">String</span><span class="cm-operator">&gt;</span> <span class="cm-variable">f</span> <span class="cm-operator">=</span> <span class="cm-variable">p</span> <span class="cm-operator">-&gt;</span> {<span class="cm-type">StringBuffer</span> <span class="cm-variable">out</span> <span class="cm-operator">=</span> <span class="cm-keyword">new</span> <span class="cm-type">StringBuffer</span>();                                     <span class="cm-variable">out</span>.<span class="cm-variable">append</span>(<span class="cm-variable">p</span>.<span class="cm-variable">getName</span>());                                     <span class="cm-variable">out</span>.<span class="cm-variable">append</span>(<span class="cm-variable">MyObject</span>.<span class="cm-variable">getStaticName</span>());                                     <span class="cm-variable">out</span>.<span class="cm-variable">append</span>(<span class="cm-string">"Constant message"</span>);                                     <span class="cm-keyword">return</span> <span class="cm-variable">out</span>.<span class="cm-variable">toString</span>();                               };</code></pre><p>Given this POJO:</p><pre><code class="language-java"><span class="cm-keyword">public</span> <span class="cm-keyword">class</span> <span class="cm-def">Triangle</span> {  <span class="cm-keyword">public</span> <span class="cm-keyword">final</span> <span class="cm-type">int</span> <span class="cm-variable">height</span>;  <span class="cm-keyword">public</span> <span class="cm-keyword">final</span> <span class="cm-type">int</span> <span class="cm-variable">base</span>;  <span class="cm-keyword">private</span> <span class="cm-type">double</span> <span class="cm-variable">area</span>;  <span class="cm-keyword">public</span> <span class="cm-type">void</span> <span class="cm-variable">setArea</span>(<span class="cm-type">double</span> <span class="cm-variable">a</span>) {    <span class="cm-variable">area</span> <span class="cm-operator">=</span> <span class="cm-variable">a</span>;  }  <span class="cm-keyword">public</span> <span class="cm-type">double</span> <span class="cm-variable">getArea</span>() {    <span class="cm-keyword">return</span> <span class="cm-variable">area</span>;  }}</code></pre><p>Write a function that sets the area of the given Triangle and returns the area as a <code>double</code>. Do not use the function type Function; there is a function type for converting an arbitrary class into a <code>double</code>, use that.</p><p>The formula for triangle area is: 1/2 * base * height</p><p>Assume valid input (base and height are both greater than 0).</p><p>A full listing of the default function types can be found at <a href="http://docs.oracle.com/javase/8/docs/api/java/util/function/package-summary.html" data-turbolinks="false" target="_blank">http://docs.oracle.com/javase/8/docs/api/java/util/function/package-summary.html</a></p><p>(Aside: you may have noticed, the function type we should have used in Part 1 is Predicate, even though we used Function.)</p>
