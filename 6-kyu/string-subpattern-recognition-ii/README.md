<h2><a href=https://www.codewars.com/kata/5a4a391ad8e145cdee0000c4/train/java target="_blank">String subpattern recognition II</a></h2><h3>6 kyu</h3><p>Similarly to the <a href="https://www.codewars.com/kata/string-subpattern-recognition-i/" data-turbolinks="false" target="_blank">previous kata</a>, you will need to return a boolean value if the base string can be expressed as the repetition of one subpattern.</p><p>This time there are two small changes:</p><ul><li>if a subpattern has been used, it will be present at least twice, meaning the subpattern has to be shorter than the original string;</li><li>the strings you will be given might or might not be created repeating a given subpattern, then shuffling the result.</li></ul><p>For example:</p><pre><code class="language-javascript"><span class="cm-string">"a"</span>    <span class="cm-operator">--&gt;</span> <span class="cm-atom">false</span> <span class="cm-comment">//no repeated shorter sub-pattern, just one character</span><span class="cm-string">"aaaa"</span> <span class="cm-operator">--&gt;</span> <span class="cm-atom">true</span>  <span class="cm-comment">//just one character repeated</span><span class="cm-string">"abcd"</span> <span class="cm-operator">--&gt;</span> <span class="cm-atom">false</span> <span class="cm-comment">//no repetitions</span><span class="cm-string">"babababababababa"</span> <span class="cm-operator">--&gt;</span> <span class="cm-atom">true</span> <span class="cm-comment">//repeated "ba"</span><span class="cm-string">"bbabbaaabbaaaabb"</span> <span class="cm-operator">--&gt;</span> <span class="cm-atom">true</span> <span class="cm-comment">//same as above, just shuffled</span></code></pre><p>Strings will never be empty and can be composed of any character (just consider upper- and lowercase letters as different entities) and can be pretty long (keep an eye on performances!).</p><p>If you liked it, go for either the <a href="https://www.codewars.com/kata/string-subpattern-recognition-i/" data-turbolinks="false" target="_blank">previous kata</a> or the <a href="https://www.codewars.com/kata/string-subpattern-recognition-iii/" data-turbolinks="false" target="_blank">next kata</a> of the series!</p>
