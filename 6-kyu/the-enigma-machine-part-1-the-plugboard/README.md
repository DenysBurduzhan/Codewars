<h2><a href=https://www.codewars.com/kata/5523b97ac8f5025c45000900/train/java target="_blank">The Enigma Machine - Part 1: The Plugboard</a></h2><h3>6 kyu</h3><p>In this series of Kata, we will be implementing a software version of the <a href="http://en.wikipedia.org/wiki/Enigma_machine" data-turbolinks="false" target="_blank">Enigma Machine</a>.</p><p>The Enigma Machine was a message enciphering/deciphering machine used during the Second World War for disguising the content of military communications.  <a href="http://en.wikipedia.org/wiki/Alan_Turing" data-turbolinks="false" target="_blank">Alan Turing</a> - the father of computing - formulated and developed concepts that are the basis of all computers in use today, he did this in response to the vital need to break those military communications.  Turing and his colleagues at <a href="http://en.wikipedia.org/wiki/Bletchley_Park" data-turbolinks="false" target="_blank">Bletchley Park</a> are generally recognised as being responsible for shortening WWII by two years and saving an estimated 22 Million lives.</p><p>The Enigma Machine consisted of a number of parts: Keyboard for input, rotors and plugboard for enciphering, and lampboard for output.</p><p>We will simulate input and output with strings, and build the rotors, plugboard and mechanism that used them in software.  As we progress the code will become more complex, so you are advised to attempt them in order.</p><p>Step 1: The <a href="http://en.wikipedia.org/wiki/Enigma_machine#Plugboard" data-turbolinks="false" target="_blank">plugboard</a></p><p>In this Kata, you must implement the plugboard.  </p><h3 id="physical-description">Physical Description</h3><p>The plugboard crosswired the 26 letters of the latin alphabet togther, so that an input into one letter could generate output as another letter.  If a wire was not present, then the input letter was unchanged.  Each plugboard came with a maximum of 10 wires, so at least six letters were not cross-wired.</p><p>For example:</p><ul><li><p>If a wire connects <code>A</code> to <code>B</code>, then an <code>A</code> input will generate a <code>B</code> output and a <code>B</code> input will generate an <code>A</code> output.</p></li><li><p>If no wire connects to <code>C</code>, then only a <code>C</code> input will generate a <code>C</code> output.</p></li></ul><h3 id="note">Note</h3><p>In the actual usage of the original Enigma Machine, punctuation was encoded as words transmitted in the stream, in our code, anything that is not in the range A-Z will be returned unchanged.</p><h3 id="kata">Kata</h3><p>The <code>Plugboard</code> class you will implement, will:</p><ol><li>Take a list of wire pairs at construction in the form of a string, with a default behaviour of no wires configured.  E.g. <code>"ABCD"</code> would wire <code>A</code> &lt;-&gt; <code>B</code> and <code>C</code> &lt;-&gt; <code>D</code>.</li><li>Validate that the wire pairings are legitimate.
